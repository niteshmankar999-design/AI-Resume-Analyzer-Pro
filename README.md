# 🚀 AI Resume Analyzer Pro

An AI-powered Resume Analysis and Job Matching platform built using Python, NLP, Streamlit, and Gemini AI.

## Features

✅ ATS Resume Scoring

✅ Resume Parsing

✅ Semantic Job Matching

✅ Skill Gap Analysis

✅ ATS Keyword Coverage

✅ AI-Powered Resume Feedback

✅ PDF Report Generation

## Project Workflow

1. Upload Resume
2. Paste Job Description
3. Analyze ATS Score
4. Calculate Job Match Score
5. Identify Missing Skills
6. Generate AI Feedback
7. Download PDF Report
